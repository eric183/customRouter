# customRouter


## SAMPLE

```javascript
  new Router().map({
     '/': function() {
        /*DO SOMETHING*/
     },
     '/sample/param/2': function(object) {
        /*DO SOMETHING WITH OBJECT LIKE: 
            console.log(object.param);
        */
     }
  })

```

server || front use, whatever you like.
